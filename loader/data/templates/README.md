# Шаблоны входных CSV-файлов

Каждый файл — заголовок + 1-2 примера. Скопируйте в `data/full/` или `data/smoke/`,
замените содержимое реальными данными и запустите загрузчик.

## Формат

Все CSV — UTF-8, разделитель «,», строка заголовка обязательна, числа с точкой.
`fetched_at` — ISO-8601 с зоной (`2026-05-18T12:00:00Z`).

## Перечень файлов и таблиц-приёмников

| Файл | Таблица |
|------|--------|
| `products.csv` | `products` |
| `product_micronutrients.csv` | `product_micronutrients` (join по `products.name`) |
| `micronutrients.csv` | `micronutrients` |
| `micronutrient_norms.csv` | `micronutrient_norms` |
| `energy_norms.csv` | `energy_norms` |

## Зависимости загрузки (порядок имеет значение)

1. `micronutrients.csv` — заводит `nutrient_id`.
2. `products.csv` — заводит продукты.
3. `product_micronutrients.csv` — связывает продукты и нутриенты.
4. `micronutrient_norms.csv` — нормы по группам.
5. `energy_norms.csv` — независимо.

CLI `loader load-all` запускает в этом порядке автоматически.

## Источники

- Продукты + нутриенты в продуктах — Скурихин И.М., Тутельян В.А. «Таблицы химического
  состава и калорийности российских продуктов питания» (2007, ISBN 978-5-94343-122-7);
  или электронная база ФИЦ питания <http://web.ion.ru/food/FD_tree_grid.aspx>.
- Нормы потребления — МР 2.3.1.0253-21 «Нормы физиологических потребностей в энергии
  и пищевых веществах для различных групп населения Российской Федерации»
  (Роспотребнадзор, 2021).
- Верхние допустимые уровни (UL) — МР 2.3.1.1915-04, для непокрытых — EFSA.
